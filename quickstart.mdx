## For Requesters

Get started as a service requester in just a few steps.

<Steps>
  <Step title="Create Your Account">
    Sign up using Google OAuth or email/password:

    ```typescript
    // Google OAuth authentication
    import { useGoogleLogin } from '@react-oauth/google';

    const login = useGoogleLogin({
      onSuccess: async (tokenResponse) => {
        // Handle successful authentication
        console.log('Access Token:', tokenResponse.access_token);
      },
      onError: () => console.log('Login Failed')
    });
    ```

    <Info>
      Google OAuth provides the fastest signup experience. You can also register manually with email and password.
    </Info>
  </Step>

  <Step title="Complete Your Profile">
    Add your phone number and location to help fixers find you:

    - **Phone Number**: Required for appointment confirmations
    - **Location**: Helps match you with nearby service providers
    - **Profile Photo**: Optional but recommended for trust

    ```typescript
    interface UserProfile {
      telefono?: string;
      ubicacion?: {
        lat?: number;
        lng?: number;
        direccion?: string;
        departamento?: string;
        pais?: string;
      };
    }
    ```
  </Step>

  <Step title="Find Services">
    Browse or search for the service you need:

    **Three Ways to Find Services:**
    1. **Browse Categories**: Choose from 24+ service categories
    2. **Advanced Search**: Use filters for location, price, and ratings
    3. **Map View**: See service providers near you on an interactive map

    ```typescript
    // Search with filters
    const { data: jobs } = useGetJobsWithFiltersQuery({
      category: 'plumbing',
      city: 'La Paz',
      minRating: 4.0,
      maxPrice: 500
    });
    ```
  </Step>

  <Step title="Book an Appointment">
    Contact a fixer and schedule your service:

    - Choose between virtual or in-person appointments
    - Select date and time
    - Provide job details and requirements
    - Confirm location (for in-person services)

    ```typescript
    interface Appointment {
      type: 'virtual' | 'presential';
      scheduledDate: string;
      meetingLink?: string; // For virtual appointments
      location?: {
        lat: number;
        lng: number;
        direccion: string;
      };
    }
    ```
  </Step>

  <Step title="Complete & Review">
    After service completion:
    
    1. **Pay**: Use wallet, QR code, or cash
    2. **Rate**: Give a 1-3 star rating
    3. **Review**: Share your experience

    <Tip>
      Good reviews help fixers build their reputation and help other requesters make informed decisions.
    </Tip>
  </Step>
</Steps>

## For Fixers

Start offering your services and accepting jobs.

<Steps>
  <Step title="Sign Up">
    Create a requester account first, then convert it to a fixer profile:

    ```typescript
    const [convertToFixer] = useConvertToFixerMutation();
    
    await convertToFixer({
      userId: user._id,
      ci: 'your-id-number'
    });
    ```

    <Note>
      All fixers start as requesters and then upgrade their account. This allows you to both offer and request services.
    </Note>
  </Step>

  <Step title="Build Your Profile">
    Complete your fixer profile with essential information:

    **Required Information:**
    - Identity Document (CI)
    - Work location and service area
    - Services you offer (select from 24+ categories)
    - Payment methods (cash, QR, credit card)
    - Vehicle information (if applicable)

    ```typescript
    interface FixerProfile {
      ci: string;
      servicios: string[];
      workLocation: {
        lat: number;
        lng: number;
        direccion: string;
      };
      metodoPago: {
        hasEfectivo?: boolean;
        qr?: boolean;
        tarjetaCredito?: boolean;
      };
      vehiculo?: {
        hasVehiculo: boolean;
        tipoVehiculo?: string;
      };
    }
    ```
  </Step>

  <Step title="Create Job Offers">
    List the services you provide:

    - Add photos of your work
    - Set competitive pricing
    - Write detailed descriptions
    - Highlight your expertise

    <Tip>
      Quality photos and detailed descriptions help attract more clients. Include before/after photos when possible.
    </Tip>
  </Step>

  <Step title="Manage Your Calendar">
    Set your availability and manage appointments:

    - View scheduled appointments
    - Accept or decline job requests
    - Update your availability
    - Manage multiple bookings

    The calendar syncs across desktop and mobile views for easy access anywhere.
  </Step>

  <Step title="Accept Jobs & Get Paid">
    Start working and earning:

    1. **Receive Requests**: Get notified of new job requests
    2. **Accept Jobs**: Review details and accept jobs that fit your schedule
    3. **Complete Work**: Provide excellent service
    4. **Get Paid**: Receive payment through your preferred method
    5. **Build Reputation**: Earn ratings and reviews

    <Card title="Payment Options" icon="wallet">
      - Wallet: Instant digital payments
      - QR Code: Quick mobile payments
      - Cash: Traditional payment method
      - Credit Card: Through Stripe integration
    </Card>
  </Step>
</Steps>
