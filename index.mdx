## What is Servineo?

Servineo is a comprehensive service marketplace platform built with Next.js that connects service requesters (customers) with skilled service providers (fixers). Whether you need a plumber, electrician, cleaner, or any other professional service, Servineo makes it easy to find, book, and pay for services in your area.

## Technology Stack

Servineo is built with modern technologies:

- **Frontend**: Next.js 15, React 19, TypeScript
- **State Management**: Redux Toolkit with Redux Logger
- **Styling**: Tailwind CSS, Material-UI, Framer Motion
- **Maps**: Leaflet and React Leaflet
- **Payments**: Stripe integration
- **Internationalization**: next-intl for multi-language support
- **Calendar**: React Big Calendar for scheduling
