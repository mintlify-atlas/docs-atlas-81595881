## Overview

Servineo features a comprehensive notification system that supports multiple notification channels (WhatsApp, Email, Appointments, Wallet), real-time updates, offline support, and role-based filtering. The system is built with performance and user experience in mind.

## Core Features

### Real-Time Polling

The system polls for new notifications every 5 seconds:

```typescript
const POLLING_INTERVAL = 5000; // 5 seconds

useEffect(() => {
  if (!isAuthenticated || !userId || !isOpen) return;

  // Initial fetch
  fetchNotifications(filter, 1, false);

  // Set up polling interval
  const intervalId = setInterval(() => {
    fetchUnreadCount();
  }, POLLING_INTERVAL);

  return () => clearInterval(intervalId);
}, [isOpen, filter, isAuthenticated, userId]);
```

### Role-Based Filtering

Notifications are filtered based on user role (Fixer vs Requester):

```typescript
const filterNotificationsByRole = (
  notifications: INotification[],
  role: 'fixer' | 'requester'
): INotification[] =>
  notifications.filter((notification) => {
    const tipoLower = notification.tipo?.trim().toLowerCase();

    if (role === 'requester') {
      // Requesters don't see wallet/balance notifications
      return !isSaldoNotification(notification) && notification.saldo === undefined;
    }

    if (tipoLower === 'desconexion 2 dias') {
      return false; // Filter out disconnection warnings
    }

    return true;
  });

const isSaldoNotification = (notification: INotification): boolean => {
  const tipoLower = notification.tipo?.toLowerCase() ?? '';
  return tipoLower.includes('saldo');
};
```

<Note>
Fixers see wallet/balance notifications, while requesters only see job-related notifications.
</Note>

### Offline Support

The notification system works offline using localStorage caching:

<CodeGroup>
```typescript Cache Management
const storageKey = (key: string) => `notifications_${userId}_${key}`;

// Load cached notifications on mount
useEffect(() => {
  const cached = localStorage.getItem(storageKey('cached_notifications'));
  if (cached) {
    try {
      const cachedData: INotification[] = JSON.parse(cached);
      const sanitized = filterNotificationsByRole(cachedData, userRole);
      setNotifications(sanitized);
    } catch (e) {
      console.error('Error loading cached notifications:', e);
    }
  }
}, [userId]);

// Save to cache after successful fetch
useEffect(() => {
  if (notifications.length > 0) {
    localStorage.setItem(
      storageKey('cached_notifications'), 
      JSON.stringify(notifications)
    );
  }
}, [notifications]);
```

```typescript Offline Queue
const [pendingSync, setPendingSync] = useState<INotification[]>([]);

const syncPendingNotifications = useCallback(async () => {
  if (pendingSync.length === 0) return;

  const currentPending = [...pendingSync];
  setPendingSync([]);

  try {
    for (const notification of currentPending) {
      await fetch(`${NOTIFICATIONS_API_URL}/mark-read`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(notification),
      });
    }
  } catch (err) {
    console.error('Error syncing notifications:', err);
    setPendingSync((prev) => [...prev, ...currentPending]);
  }
}, [pendingSync]);

// Sync when back online
useEffect(() => {
  if (isOnline && pendingSync.length > 0) {
    syncPendingNotifications();
  }
}, [isOnline, syncPendingNotifications]);
```
</CodeGroup>

### Read/Unread Tracking

The system tracks read notifications locally and syncs with the server:

```typescript
const [readNotifications, setReadNotifications] = useState<Set<string>>(new Set());

// Load read status from localStorage
useEffect(() => {
  const savedRead = localStorage.getItem(storageKey('read_notifications'));
  if (savedRead) {
    try {
      const readIds = JSON.parse(savedRead);
      setReadNotifications(new Set(readIds));
    } catch (e) {
      console.error('Error loading read notifications:', e);
    }
  }
}, [userId]);

// Calculate unread count
const unreadCount = notifications.filter(
  (n) => !(n.leido || readNotifications.has(n._id))
).length;

// Mark notification as read
const markAsRead = (notificationId: string) => {
  setReadNotifications((prev) => {
    const updated = new Set(prev);
    updated.add(notificationId);
    
    // Save to localStorage
    localStorage.setItem(
      storageKey('read_notifications'),
      JSON.stringify(Array.from(updated))
    );
    
    return updated;
  });
};
```

## Pagination

Infinite scroll pagination for loading more notifications:

```typescript
const NOTIFICATIONS_PER_PAGE = 20;
const [page, setPage] = useState(1);
const [hasMore, setHasMore] = useState(true);
const [loadingMore, setLoadingMore] = useState(false);

const loadMore = useCallback(() => {
  if (loadingMore || !hasMore) return;
  fetchNotifications(filter, page + 1, true);
}, [filter, page, loadingMore, hasMore]);

// Intersection Observer for infinite scroll
const observerRef = useRef<IntersectionObserver | null>(null);

useEffect(() => {
  const sentinel = document.getElementById('notification-sentinel');
  if (!sentinel) return;

  observerRef.current = new IntersectionObserver(
    (entries) => {
      if (entries[0].isIntersecting && hasMore && !loadingMore) {
        loadMore();
      }
    },
    { threshold: 0.5 }
  );

  observerRef.current.observe(sentinel);

  return () => {
    if (observerRef.current) {
      observerRef.current.disconnect();
    }
  };
}, [hasMore, loadingMore, loadMore]);
```
