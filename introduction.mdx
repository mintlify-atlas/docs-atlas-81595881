## Welcome to Servineo

Servineo is a comprehensive platform that connects service requesters with skilled professionals (fixers) across a wide range of services. Whether you need a plumber, electrician, carpenter, or any other home service professional, Servineo makes it easy to find, contact, and hire the right person for the job.

## User Roles

Servineo supports three distinct user roles:

### Requester
Clients who need services. Requesters can:
- Browse and search for fixers by service type and location
- View fixer profiles, ratings, and experience
- Create job requests with detailed descriptions and pricing
- Schedule appointments with fixers
- Track job progress and make payments
- Leave ratings and reviews

### Fixer
Service professionals who offer their expertise. Fixers can:
- Create detailed profiles showcasing services and experience
- Receive job requests from requesters
- Manage appointments and schedules
- Accept or decline job offers
- Receive payments through multiple methods
- Build reputation through ratings and reviews

### Admin
Platform administrators who manage the overall system and ensure quality service.

## Key Features

### Authentication & Profile Management
Servineo offers flexible authentication options:

```typescript
interface IUser {
  _id?: string;
  id?: string;
  name: string;
  email: string;
  url_photo?: string;
  role: 'requester' | 'fixer' | 'admin';
  
  authProviders?: Array<{
    provider: string;
    providerId: string;
    password?: string;
  }>;
  
  telefono?: string;
  ubicacion?: {
    lat?: number;
    lng?: number;
    direccion?: string;
    departamento?: string;
    pais?: string;
  };
}
```

<Tip>
Users can sign up using Google OAuth or manual registration with email and password. Both authentication methods are fully supported.
</Tip>

### Location-Based Services

The platform uses geographic coordinates to match requesters with nearby fixers:

```typescript
interface CreateJobRequestPayload {
  title: string;
  description: string;
  location: {
    type: 'Point';
    coordinates: [string, string]; // [longitude, latitude]
  };
  startTime: string;
  endTime: string;
  price: string | number;
  fixerId: string;
  requesterId: string;
}
```

### Multi-Language Support

Servineo is built with internationalization using `next-intl`, supporting multiple languages for a global audience.

## Technology Stack

Servineo leverages modern web technologies:

- **Frontend**: Next.js 15, React 19, TypeScript
- **State Management**: Redux Toolkit with RTK Query
- **UI Components**: Material-UI, Radix UI, Tailwind CSS
- **Maps**: Leaflet and React Leaflet
- **Payments**: Stripe integration
- **Authentication**: JWT with Google OAuth support
- **Forms**: React Hook Form with Zod validation
