## Get Appointments by Fixer

Retrieve all appointments for a specific fixer.

### Endpoint

```
GET /api/appointments/fixer/{fixerId}
```

### Path Parameters

<ParamField path="fixerId" type="string" required>
  The unique identifier of the fixer
</ParamField>

### Response

Array of appointment objects.

<ResponseField name="[].appointment" type="object">
  Individual appointment object
  
  <Expandable title="Appointment Object">
    <ResponseField name="_id" type="string">
      Unique appointment identifier
    </ResponseField>
    
    <ResponseField name="id_fixer" type="string">
      Fixer's unique identifier
    </ResponseField>
    
    <ResponseField name="id_requester" type="string">
      Requester's unique identifier
    </ResponseField>
    
    <ResponseField name="selected_date" type="object">
      Selected appointment date in MongoDB format
      ```json
      { "$date": "2024-03-15T00:00:00.000Z" }
      ```
    </ResponseField>
    
    <ResponseField name="current_requester_name" type="string">
      Name of the person requesting the service
    </ResponseField>
    
    <ResponseField name="appointment_type" type="string">
      Type of appointment/service
    </ResponseField>
    
    <ResponseField name="appointment_description" type="string">
      Detailed description of the work needed
    </ResponseField>
    
    <ResponseField name="link_id" type="string">
      Link identifier for related resources
    </ResponseField>
    
    <ResponseField name="current_requester_phone" type="string">
      Contact phone number of the requester
    </ResponseField>
    
    <ResponseField name="starting_time" type="object">
      Appointment start time
      ```json
      { "$date": "2024-03-15T09:00:00.000Z" }
      ```
    </ResponseField>
    
    <ResponseField name="finishing_time" type="object">
      Appointment end time
      ```json
      { "$date": "2024-03-15T12:00:00.000Z" }
      ```
    </ResponseField>
    
    <ResponseField name="schedule_state" type="string">
      Current state: `booked`, `cancelled`, `completed`, `in_progress`
    </ResponseField>
    
    <ResponseField name="display_name_location" type="string">
      Human-readable location name
    </ResponseField>
    
    <ResponseField name="lat" type="string | null">
      Latitude coordinate of the appointment location
    </ResponseField>
    
    <ResponseField name="lon" type="string | null">
      Longitude coordinate of the appointment location
    </ResponseField>
    
    <ResponseField name="cancelled_fixer" type="boolean">
      Whether the appointment was cancelled by the fixer
    </ResponseField>
    
    <ResponseField name="reprogram_reason" type="string">
      Reason for rescheduling (if applicable)
    </ResponseField>
    
    <ResponseField name="createdAt" type="object">
      Appointment creation timestamp
    </ResponseField>
    
    <ResponseField name="updatedAt" type="object">
      Last update timestamp
    </ResponseField>
    
    <ResponseField name="__v" type="number">
      Version key for MongoDB
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl https://api.servineo.com/api/appointments/fixer/fixer123 \
  -H "Content-Type: application/json"
```

```typescript TypeScript
import { getAppointmentsByFixerId } from '@/services/appointments';

const appointments = await getAppointmentsByFixerId('fixer123');
console.log(`Found ${appointments.length} appointments`);
```

```javascript JavaScript
const fixerId = 'fixer123';

const response = await fetch(
  `https://api.servineo.com/api/appointments/fixer/${fixerId}`,
  {
    method: 'GET',
    headers: {
      'Content-Type': 'application/json'
    },
    cache: 'no-store'
  }
);

const appointments = await response.json();
```

</CodeGroup>

## Get Map Locations (Admin)

Retrieve all appointment locations for map visualization (admin only).

### Endpoint

```
GET /api/admin/map-locations
```

### Authorization

<Warning>
  This endpoint requires admin authentication.
</Warning>

### Response

<ResponseField name="[].location" type="object">
  Map location object
  
  <Expandable title="Location Properties">
    <ResponseField name="_id" type="string">
      Appointment ID
    </ResponseField>
    
    <ResponseField name="lat" type="string">
      Latitude coordinate
    </ResponseField>
    
    <ResponseField name="lon" type="string">
      Longitude coordinate
    </ResponseField>
    
    <ResponseField name="schedule_state" type="string">
      State: `booked` or `cancelled`
    </ResponseField>
    
    <ResponseField name="fixerName" type="string">
      Name of the assigned fixer
    </ResponseField>
    
    <ResponseField name="current_requester_name" type="string">
      Name of the requester
    </ResponseField>
    
    <ResponseField name="starting_time" type="string">
      ISO 8601 formatted start time
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl https://api.servineo.com/api/admin/map-locations \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json"
```

## Get Tracking Metrics (Admin)

Retrieve appointment statistics and metrics.

### Endpoint

```
GET /api/admin/metrics
```

### Query Parameters

<ParamField query="startDate" type="string">
  Filter start date (ISO 8601 format)
</ParamField>

<ParamField query="endDate" type="string">
  Filter end date (ISO 8601 format)
</ParamField>

### Response

<ResponseField name="total" type="number">
  Total number of appointments
</ResponseField>

<ResponseField name="active" type="number">
  Number of active/booked appointments
</ResponseField>

<ResponseField name="cancelled" type="number">
  Number of cancelled appointments
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl "https://api.servineo.com/api/admin/metrics?startDate=2024-03-01&endDate=2024-03-31" \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json"
```

```typescript TypeScript
import { useGetTrackingMetricsQuery } from '@/app/redux/services/trackingAppointmentsApi';

const { data: metrics } = useGetTrackingMetricsQuery({
  startDate: '2024-03-01',
  endDate: '2024-03-31'
});
```

</CodeGroup>

### Example Response

```json
{
  "total": 150,
  "active": 120,
  "cancelled": 30
}
```

## Appointment States

Possible values for `schedule_state`:

| State | Description |
|-------|-------------|
| `booked` | Appointment is confirmed and scheduled |
| `cancelled` | Appointment has been cancelled |
| `completed` | Service has been completed |
| `in_progress` | Fixer is currently working on the job |

## Error Responses

### No Appointments Found (404)

```json
{
  "success": false,
  "error": "No appointments found for fixer"
}
```

### Unauthorized (401)

  "error": "Unauthorized - Admin access required"
}
