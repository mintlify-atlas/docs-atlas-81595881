## Overview

Retrieve all fixers who provide a specific type of service/job.

## Endpoint

```
GET /api/jobs/{jobType}/fixers
```

## Path Parameters

<ParamField path="jobType" type="string" required>
  The type of job/service (e.g., "plomeria", "electricidad", "pintura")
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of fixer objects who provide the specified service
  
  <Expandable title="Fixer Object">
    <ResponseField name="id" type="string">
      Fixer's unique identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Fixer's full name
    </ResponseField>
    
    <ResponseField name="city" type="string">
      City where the fixer operates
    </ResponseField>
    
    <ResponseField name="rating" type="number">
      Average rating from 0 to 5
    </ResponseField>
    
    <ResponseField name="avatar" type="string">
      URL to the fixer's profile photo
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Fixer's bio or description of services
    </ResponseField>
    
    <ResponseField name="phone" type="string">
      Contact phone number
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Contact email address
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL - Plomeria
curl https://api.servineo.com/api/jobs/plomeria/fixers \
  -H "Content-Type: application/json"
```

```bash cURL - Electricidad
curl https://api.servineo.com/api/jobs/electricidad/fixers \
  -H "Content-Type: application/json"
```

```typescript TypeScript
import { fixerService } from '@/services/fixerService';

// Get all plumbers
const plumbers = await fixerService.getFixersByJobType('plomeria');

// Get all electricians
const electricians = await fixerService.getFixersByJobType('electricidad');

// Get all painters
const painters = await fixerService.getFixersByJobType('pintura');
```

```javascript JavaScript
const jobType = 'plomeria';

const response = await fetch(
  `https://api.servineo.com/api/jobs/${jobType}/fixers`,
  {
    method: 'GET',
    headers: {
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
const fixers = data.data;
```

</CodeGroup>

## Filtering and Sorting

The API returns fixers sorted by rating (highest first) by default. You can implement additional client-side filtering:

```typescript
// Filter by minimum rating
const topFixers = fixers.filter(fixer => fixer.rating >= 4.5);

// Filter by city
const localFixers = fixers.filter(fixer => fixer.city === 'Asunción');

// Sort by name
const sortedByName = [...fixers].sort((a, b) => 
  a.name.localeCompare(b.name)
);
```

## Get Fixer by ID

If you need detailed information about a specific fixer:

### Endpoint

```
GET /api/fixers/{id}
```

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the fixer
</ParamField>

### Example Request

## Get All Fixers

Retrieve all fixers regardless of job type:

### Endpoint

```
GET /api/fixers
```

### Example Request

<CodeGroup>

```bash cURL
curl https://api.servineo.com/api/fixers \
  -H "Content-Type: application/json"
```

```typescript TypeScript
import { fixerService } from '@/services/fixerService';

const allFixers = await fixerService.getFixers();
console.log(allFixers);
```

</CodeGroup>
