## Create Job Offer

Create a new job offer as a fixer.

### Endpoint

```
POST /api/job-offers
```

### Request Body

<ParamField body="formData" type="FormData" required>
  Multipart form data containing job offer details and images
  
  <Expandable title="FormData Fields">
    <ParamField body="title" type="string" required>
      Job offer title
    </ParamField>
    
    <ParamField body="description" type="string" required>
      Detailed description of the service offered
    </ParamField>
    
    <ParamField body="category" type="string" required>
      Service category (e.g., "plomeria", "electricidad")
    </ParamField>
    
    <ParamField body="price" type="number" required>
      Service price in local currency
    </ParamField>
    
    <ParamField body="fixerId" type="string" required>
      Fixer's unique identifier
    </ParamField>
    
    <ParamField body="images" type="file[]">
      Array of image files to showcase the service
    </ParamField>
  </Expandable>
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.servineo.com/api/job-offers \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -F "title=Reparaciones de plomería" \
  -F "description=Ofrezco servicios profesionales de plomería" \
  -F "category=plomeria" \
  -F "price=150000" \
  -F "fixerId=fixer123" \
  -F "images=@/path/to/image1.jpg" \
  -F "images=@/path/to/image2.jpg"
```

formData.append('fixerId', 'fixer123');
formData.append('images', imageFile1);

## Update Job Offer

Update an existing job offer.

### Endpoint

```
PATCH /api/job-offers/{jobId}
```

### Path Parameters

<ParamField path="jobId" type="string" required>
  The unique identifier of the job offer
</ParamField>

### Request Body

<ParamField body="formData" type="Partial<FormData>" required>
  Partial form data with fields to update
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X PATCH https://api.servineo.com/api/job-offers/offer123 \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -F "title=Reparaciones de plomería - Actualizado" \
  -F "price=180000"
```

```typescript TypeScript
import { useUpdateJobMutation } from '@/app/redux/services/jobApi';

const [updateJob] = useUpdateJobMutation();

const formData = new FormData();
formData.append('title', 'Reparaciones de plomería - Actualizado');
formData.append('price', '180000');

const result = await updateJob({
  jobId: 'offer123',
  formData
});
```

</CodeGroup>

### Response

```json
{
  "_id": "offer123",
  "title": "Reparaciones de plomería - Actualizado",
  "price": 180000,
  "updatedAt": "2024-03-14T16:00:00.000Z"
}
```

### Path Parameters

<ParamField path="jobId" type="string" required>

</ParamField>

### Example Request

<CodeGroup>

## Delete Job Offer

Delete a job offer permanently.

### Endpoint

```
DELETE /api/job-offers/{jobId}
```

### Path Parameters

<ParamField path="jobId" type="string" required>
  The unique identifier of the job offer
</ParamField>

### Request Body

<ParamField body="fixerId" type="string" required>
  The fixer ID to verify ownership
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X DELETE https://api.servineo.com/api/job-offers/offer123 \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"fixerId": "fixer123"}'
```

```typescript TypeScript
import { useDeleteJobMutation } from '@/app/redux/services/jobApi';

const [deleteJob] = useDeleteJobMutation();

const result = await deleteJob({
  jobId: 'offer123',
  fixerId: 'fixer123'
});
```

</CodeGroup>

### Response

```json
{
  "message": "Job offer deleted successfully"
}
```

## Create Job Request

Create a new job request as a requester.

### Endpoint

```
POST /api/jobrequests
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Request Body

<ParamField body="title" type="string" required>
  Job request title/motive
</ParamField>

<ParamField body="description" type="string" required>
  Detailed description of the work needed
</ParamField>

<ParamField body="location" type="object" required>
  Job location
  
  <Expandable title="Location Properties">
    <ParamField body="type" type="string" required>
      Must be "Point"
    </ParamField>
    
    <ParamField body="coordinates" type="array" required>
      Array with [longitude, latitude] as strings
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="startTime" type="string" required>
  Start time in ISO 8601 format
</ParamField>

<ParamField body="endTime" type="string" required>
  End time in ISO 8601 format
</ParamField>

<ParamField body="price" type="string | number" required>
  Suggested price for the job
</ParamField>

<ParamField body="fixerId" type="string" required>
  ID of the fixer being requested
</ParamField>

<ParamField body="requesterId" type="string" required>
  ID of the user making the request
</ParamField>

<ParamField body="appointmentId" type="string">
  Optional appointment ID if linked to an appointment
</ParamField>

<ParamField body="offerId" type="string">
  Optional offer ID if responding to a job offer
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.servineo.com/api/jobrequests \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Reparación de fuga de agua",
    "description": "Tengo una fuga en la cañería de la cocina",
    "location": {
      "type": "Point",
      "coordinates": ["-57.5759", "-25.2637"]
    },
    "startTime": "2024-03-15T09:00:00.000Z",
    "endTime": "2024-03-15T12:00:00.000Z",
    "price": 150000,
    "fixerId": "fixer123",
    "requesterId": "requester456"
  }'
```

```typescript TypeScript
import { createJobRequest } from '@/services/job-requests';

  price: 150000,

  requesterId: 'requester456'

```

</CodeGroup>
