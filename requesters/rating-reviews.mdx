## Rating Overview

After completing a service, you can rate your experience and leave a review. Your feedback helps other requesters make informed decisions and helps fixers improve their services.

## Rating System

Servineo uses a **1-3 star rating system** for job completion:

  <Card title="1 Star" icon="star">
    Poor service quality
  </Card>
  <Card title="2 Stars" icon="star">
    Average service quality
  </Card>
  <Card title="3 Stars" icon="star">
    Excellent service quality
  </Card>
</CardGroup>

<Note>
  The rating system uses 1-3 stars for completed jobs, while the search filter supports 1-5 stars with decimal precision for fixer overall ratings.
</Note>

Ratings are stored with the following information:

```typescript
interface FixerRating {

}

type RatingResponse = {
  _id: string;
```

## Viewing Ratings

### Your Submitted Ratings

View all ratings you've given to fixers in your profile section.

### Fixer Rating Details

When viewing a fixer's profile, you can see all their ratings:

```typescript
function useFixerRatings(fixerId: string, pollInterval: number = 5000) {
  const [ratings, setRatings] = useState<FixerRating[]>([]);
  const [isLoading, setIsLoading] = useState(true);
  
  const url = fixerId ? apiUrl(`/api/ratings.details/${fixerId}`) : null;
  
  const load = async () => {
    const data = await jsonFetcher<RatingResponse[]>(url);
    const mapped = data.map((d) => ({
      id: d._id,
      requester: d.title || 'Unknown',
      score: d.rating as 1 | 2 | 3,
      comment: d.comment || '',
      createdAt: d.createdAt,
    }));
    setRatings(mapped);
  };
  
  return { ratings, isLoading, error, refresh: load };
}
```

## Star Rating Component

The visual star display:

```tsx
export function StarRating({
  value,
  size = 18,
  srLabel,
}: {
  value: 1 | 2 | 3;
  size?: number;
  srLabel?: string;
}) {
  return (
    <div className="inline-flex gap-1" aria-label={srLabel ?? `${value} of 3 stars`}>
      {Array.from({ length: 3 }).map((_, i) => {
        const filled = i < value;
        return (
          <svg
            key={i}
            width={size}
            height={size}
            viewBox="0 0 24 24"
            style={
              filled
                ? { fill: 'var(--highlight)' }
                : { fill: 'transparent', stroke: 'var(--highlight)' }
            }
          >
            <path d="M12 17.27 18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
          </svg>
        );
      })}
    </div>
  );
}
```

```tsx
export function RatingDetailsList({ ratings, error }: {
```

## Rating Statistics

Fixers' overall ratings are calculated from all submitted reviews:

- **Average Rating**: Mean of all ratings (displayed as 1.0 - 5.0)
- **Total Reviews**: Count of all submitted ratings
- **Recent Ratings**: Latest reviews shown first
- **Rating Distribution**: Breakdown by star count

## Impact on Fixers

Your ratings directly affect:

- **Search Ranking**: Higher-rated fixers appear first in search results
- **Profile Visibility**: Star ratings displayed prominently on profiles
- **Trust Building**: Reviews help build credibility
- **Service Improvement**: Feedback helps fixers enhance their services

<Warning>
  Ratings cannot be edited or deleted after submission. Please review carefully before submitting.
</Warning>

## Dispute Resolution

If you believe a rating was unfair or submitted in error:

1. Contact Servineo support
2. Provide evidence of the issue
3. Support team will review the case
4. Resolution typically takes 3-5 business days
