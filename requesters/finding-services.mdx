## Search Overview

Servineo provides a powerful advanced search system to help you find the right fixer for your needs. The search interface includes multiple filters and sorting options.

## Advanced Search Features

### Search by Category

Use the category dropdown to filter fixers by service tags:

```typescript
// Category filter implementation
```

### Filter by Rating

Find highly-rated fixers using the star rating filter:

<Accordion title="How the Rating Filter Works">
  The rating system allows you to filter fixers by their average rating from 1.0 to 5.0 stars with decimal precision.

  ```typescript
  // Rating filter with decimal precision
  const handleSubScaleClick = (subScale: number) => {
    if (selectedMainStar && onChange) {
      const decimalValue = subScale === 0 
        ? selectedMainStar 
        : selectedMainStar + subScale / 10;
      onChange(decimalValue);
    }
  };
  ```

  Reference: `src/Components/Adv-search/CalificacionEstrella.tsx:53`
</Accordion>

**Rating Selection:**

1. Click on a star (1-5) to open the precision selector
2. Choose a decimal value (e.g., 4.0, 4.1, 4.2, etc.)
3. Fixers with ratings at or above your selection will be shown

<Tip>
  Use decimal ratings (e.g., 4.5) for more precise filtering. This helps you find fixers with consistently high ratings.
</Tip>

## Saving Search Filters

Your search preferences are automatically saved:

```typescript
// Filters persist in URL parameters
const sp = new URLSearchParams(window.location.search);
const urlTags = sp.getAll('tags') || [];

// Restore filters on page load
if (urlTags.length > 0) {
  setSelectedCategories(urlTags);
  onFilterChange?.({ categories: urlTags });
}
```

Reference: `src/Components/Adv-search/DropdownList.tsx:53`

<Tip>
  Bookmark your search URL to quickly access the same filtered results later.
</Tip>

## Search Best Practices

1. **Start Broad**: Begin with category selection, then refine with other filters
2. **Check Ratings**: Look for fixers with 4.0+ ratings and multiple completed jobs
3. **Verify Location**: Ensure the fixer serves your area before booking
4. **Read Reviews**: Check detailed ratings and comments from other requesters
5. **Compare Options**: View multiple profiles before making a decision
